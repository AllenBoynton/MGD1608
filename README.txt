# MGD-1608Mobile Game DesignAllen Boyntonhttps://github.com/AllenBoynton/MGD-16083 Sprites - I have more than that but the Bi-Plane, Bullet and Star2 Sound Effects - Sound of plane, Background music1 Type of input control (tap) - Release of tap fires bullets2 Game events - Tapping the star will produce a sound, Tapping the screen will fire the bullet and produce a machine gun firing sound. When the bullet hits the star the star will produce a sound.
